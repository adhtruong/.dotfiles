# .dotfiles

Repository for managing development environment configuration.

## Apps

Configuration included:

- Brew applications
- VSCode setup - emulating nvim motions
- Terminal/tmux/shell setup

## Setup

Requires `brew`

```bash
./install.sh
make all
./obsidian/sync.sh import
./preferences import
```

## Local Configuration

- `zsh/.zshrc.local`
- `git/.gitconfig_local`
