# .dotfiles

Repo for managing configuration.

## Apps

Configuration included

- Brew apps
- VSCode set up - emulating nvim motions
- Terminal/tmux/shell set up

## Set up

Requires `brew`

```bash
./install.sh
make all
./obsidian/sync.sh import
./preferences import
```

Local configuration

- `zsh/.zshrc.local`
- `git/.gitconfig_local`
